## Running the Project with Docker Compose

To run the project using Docker Compose, follow these steps:

1. **Build and start the Docker containers:**

   ```sh
   docker-compose up --build
   ```

2. **Open your web browser and navigate to:**

   ```
   http://localhost
   ```
